
                                             
                                             Mise en ligne du projet

  **Installation du projet**
  
Pour lancer le programme:

>npm run install

Démarrer le projet:

>npm run launch


Créer un fichier config.json dans Front/src:

Copier ce contenu:
>{
  "urlServer": "localhost",
  "DB":{
    "user": "appUser",
    "Pass": "appUser",
    "Database": "FreelanceVision",
    "PORT": "4080"
  }
}
